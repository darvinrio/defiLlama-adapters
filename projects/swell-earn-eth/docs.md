# Understanding `sumTokens2`

This is to understand the `sumTokens2` function :salute:

* `tokensAndOwners` -> array of array containing the token address and its respective owner
  * eg: `[]
* `uniV3nftsAndOwners` -> array of array containing the NFT manager and the NFT owner
* `uniV3ExtraConfig` -> specify if the NFT config needs to be over-ridden
  * `nftIdFetcher` -> helps to point to alternative NFT manager contracts
